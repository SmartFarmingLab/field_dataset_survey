Field Dataset Survey

Agricultural Computer Vision Dataset Survey

A curated catalogue of high-quality RGB image datasets for natural field scenes.

Overview

This project focuses on providing access to carefully selected datasets for computer vision tasks in agriculture. The collection includes:

  • Weed detection and classification
  • Disease and pest detection
  • Seedling and crop detection
  • Plant growth stage detection
  • Phenotyping
